Data Visualisation - Project

Table Tennis Rubber

Purpose of this project is to display skills, learned during labotory exercises, and knowledge, gathered in lectures, about data visualisation using Data Driven Documents library powered by JavaScript. This project includes displaying data using basic charts like: Bar chart and Donut chart, but also some more complex charts like Raddar chart. It also shows importance of choosing correct chart and colors to simplfy understanding of the data.

Data features

Speed

Rating of the power generated by the rubber or blade.

Spin

Amount of ball rotation generated by the rubber.

Control

Refers to how easy it is to control the ball using this equipment.

Tackiness

Refers to the level of stickiness of the rubber surface.

Sponge Hardness

Refers to the density of the sponge. This is measured in degrees.

Gears

Refers to a rubber's ability to produce a wide range of speeds.

Throw Angle

Angle that the ball is projected from strokes like the loop.

Consistency

How consistent the product is. Lower quality control may produce lower consistency.

Durability

Refers to how long a product lasts.

Weight

Mass of the rubber.

Overall

Refers to the overall performance rating.

Price

Cost of the rubber in dollars.
